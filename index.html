Workshop Details

This two-day workshop introduces participants to modern methods for accessing and analysing large-scale Earth Observation datasets using cloud-native geospatial technologies. Participants will explore tiled data formats, Earth Observation data cubes and scalable Python workflows that enable efficient analysis of satellite imagery without requiring large local storage.
